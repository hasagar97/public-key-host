Public Key Host

Welcome! This page hosts public keys for secure authentication and verification.

Available Public Keys

  • SSH RSA Key: ssh-rsa.pub

Usage Instructions

Adding Keys to authorized_keys

To add a key to your ~/.ssh/authorized_keys file, you can use:

curl https://hasagar97.github.io/public-key-host/keys/ssh-rsa.pub >> ~/.ssh/authorized_keys

Direct Key URL

You can reference keys directly at:

https://hasagar97.github.io/public-key-host/keys/ssh-rsa.pub

About

This repository uses GitHub Pages to host public keys. All keys are publicly accessible via HTTPS.
